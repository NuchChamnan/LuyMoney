# Prompt៖ ការពិពណ៌នាអំពីកម្មវិធី "Luy Money"

## ទិដ្ឋភាពទូទៅ
**Luy Money** គឺជាកម្មវិធីទូរស័ព្ទ (Flutter, ស្ថាបត្យកម្ម GetX) សម្រាប់ផ្តល់មាតិកាអប់រំផ្នែកហិរញ្ញវត្ថុ វិនិយោគ និងធុរកិច្ច (វីដេអូ + អត្ថបទ) ក្រោមទម្រង់សមាជិកភាពបង់ប្រាក់ (Subscription)។ កម្មវិធីភ្ជាប់ជាមួយ Firebase (Auth, Firestore, Storage, Analytics, FCM) និងគាំទ្រ ៣ ភាសា (អង់គ្លេស ខ្មែរ ចិន)។

---

## ១. ការផ្ទៀងផ្ទាត់អត្តសញ្ញាណ (Authentication)
- ចូលប្រើ និងចុះឈ្មោះគណនីដោយប្រើអ៊ីមែល/ពាក្យសម្ងាត់ (Firebase Auth + Firestore profile)
- មុខងារ "ចងចាំខ្ញុំ" និងការចូលប្រើដោយផ្ទៀងផ្ទាត់ជីវមាត្រ (Biometric/Fingerprint)
- ភ្លេចពាក្យសម្ងាត់ — ផ្ញើតំណកំណត់ពាក្យសម្ងាត់ឡើងវិញតាមអ៊ីមែល
- ផ្ទៀងផ្ទាត់ទម្រង់បញ្ចូល (អ៊ីមែល ពាក្យសម្ងាត់យ៉ាងតិច ៨ តួ ការផ្ទៀងផ្ទាត់ពាក្យសម្ងាត់ត្រូវគ្នា ឈ្មោះ)
- បន្ទាប់ពីចូលប្រើ ប្រព័ន្ធនាំទៅទំព័រសមស្របតាមតួនាទី (admin → ទំព័រគ្រប់គ្រង, user → ទំព័រដើម)
- ចេញពីគណនី (Sign out) ជាមួយប្រអប់បញ្ជាក់

## ២. ទំព័រដើម (Home)
- បង្ហាញវីដេអូ និងអត្ថបទថ្មីៗ (កំពូល ៥ មុខវិញ្ញាសា)
- បង្ហាញចំនួនប្រភេទមាតិកា (categories)
- បង្ហាញស្ថានភាពសមាជិកភាព៖ កំពុងប្រើ ជិតផុតកំណត់ ឬចំនួនថ្ងៃនៅសល់
- រុករកដោយ Bottom Navigation Bar

## ៣. មាតិកា — វីដេអូ និងអត្ថបទ (Content)
- រាយបញ្ជីវីដេអូ/អត្ថបទពី Firestore តាមកាលបរិច្ឆេទចុះផ្សាយ
- ច្រោះតាមប្រភេទ (Category Filter): ហិរញ្ញវត្ថុ វិនិយោគ ផ្លូវចិត្ត ការជួញដូរ សន្សំ ធុរកិច្ច...
- ស្វែងរកមាតិកាតាមចំណងជើង ប្រភេទ ឬខ្លឹមសារសង្ខេប (មាន debounce ៣០០ms)
- ចំណាំ/រក្សាទុក (Bookmark) វីដេអូ និងអត្ថបទ ទុកមើលនៅពេលក្រោយ
- មាតិកាមួយចំនួនជាប់សោ "Premium" — ត្រូវការសមាជិកភាពសកម្មទើបអាចមើលបាន
- ទំព័រលម្អិតវីដេអូ៖ លេង YouTube player បង្ហាញស្ថិតិ (ចំនួនមើល រយៈពេល កាលបរិច្ឆេទ) ការពណ៌នា Like/Save/More និងវីដេអូដែលពាក់ព័ន្ធ
- ទំព័រលម្អិតអត្ថបទ៖ អានមាតិកាពេញលេញ និងចំណាំ

## ៤. សមាជិកភាព និងការទូទាត់ប្រាក់ (Subscription & Payment)
- គម្រោងជាវ ៤ ប្រភេទ៖ ប្រចាំខែ ប្រចាំត្រីមាស ប្រចាំ ៦ ខែ និងប្រចាំឆ្នាំ (មានស្លាក "ពេញនិយម")
- ទូទាត់ប្រាក់តាម ABA Pay (KHQR) — ស្កេន QR ថតវិក័យប័ត្រ រួចផ្ញើទៅ Telegram Admin (@Noch_Chamnan) ដើម្បីឱ្យបុគ្គលិកបញ្ជាក់ និងបើកដំណើរការសមាជិកភាពដោយដៃ
- កូដបញ្ចុះតម្លៃ (Promo Code): ផ្ទៀងផ្ទាត់ភាពត្រឹមត្រូវ កាលបរិច្ឆេទផុតកំណត់ ចំនួនដងប្រើប្រាស់អតិបរមា និងគម្រោងដែលអាចអនុវត្តបាន គិតបញ្ចុះជាភាគរយ ឬចំនួនថេរ
- ប្រព័ន្ធណែនាំមិត្តភ័ក្តិ (Referral): បង្កើតកូដផ្ទាល់ខ្លួនដោយស្វ័យប្រវត្តិ ចែករំលែកតាមតំណ ប្រើកូដមិត្តភ័ក្តិ (ការពារការប្រើខ្លួនឯង/ប្រើច្រើនដង) ហើយអ្នកណែនាំទទួលបានប្រាក់រង្វាន់បន្ថែម ៣០ ថ្ងៃលើសមាជិកភាពនៅពេលមិត្តភ័ក្តិទិញសេវាលើកដំបូង

## ៥. ការគ្រប់គ្រងគណនី (Settings & Profile)
- កែប្រែឈ្មោះ លេខទូរស័ព្ទ និងផ្លាស់ប្តូររូបភាពប្រវត្តិរូប (base64, ≤500KB)
- ប្តូរពាក្យសម្ងាត់ (តម្រូវឱ្យផ្ទៀងផ្ទាត់ម្តងទៀត)
- លុបគណនី (លុបទិន្នន័យក្នុង Firestore ផងដែរ)
- បិទ/បើកការជូនដំណឹង៖ ការរំលឹកសមាជិកភាព ការជូនដំណឹងមាតិកាថ្មី សារផ្សព្វផ្សាយ
- វាយតម្លៃកម្មវិធី បើកគោលការណ៍ឯកជនភាព លក្ខខណ្ឌប្រើប្រាស់ FAQ និង Telegram សម្រាប់ជំនួយ
- ប្តូរភាសា៖ អង់គ្លេស ខ្មែរ ឬចិន (រក្សាទុកជម្រើស)

## ៦. ការគាំទ្រអតិថិជន (Support)
- ជជែកជាមួយក្រុមការងារតាមម៉ោងពិតប្រាកដ (Real-time chat តាម Firestore)
- ផ្ញើសារជាអក្សរ និងរូបភាព (តាមរយៈ Firebase Storage)
- បើកប៉ុស្តិ៍ Telegram ជំនួយដោយផ្ទាល់

## ៧. ផ្នែកគ្រប់គ្រង (Admin Dashboard)
ទំព័រគ្រប់គ្រងសម្រាប់គណនីដែលមានតួនាទី admin មាន ៥ ផ្នែកធំៗ៖
- **អ្នកប្រើប្រាស់**: ស្វែងរក/ច្រោះ (សកម្ម ផុតកំណត់ ឥតគិតថ្លៃ) បន្ថែមថ្ងៃសមាជិកភាព បិទដំណើរការ លុប តម្លើង/បន្ទាបតួនាទី admin និងផ្ញើអ៊ីមែលកំណត់ពាក្យសម្ងាត់ឡើងវិញ
- **មាតិកា**: បន្ថែម/លុបវីដេអូ និងអត្ថបទ គ្រប់គ្រងប្រភេទមាតិកា (បន្ថែម កែប្រែ លុប)
- **ស្ថិតិវិភាគ**: ចំនួនអ្នកប្រើសរុប/សកម្ម/ផុតកំណត់/ឥតគិតថ្លៃ ប្រាក់ចំណូលប្រចាំខែ និងសរុប ក្រាហ្វប្រាក់ចំណូល ៦ ខែចុងក្រោយ
- **ការជូនដំណឹង**: តែងនិងផ្ញើការជូនដំណឹងជាមួយអ្នកប្រើគោលដៅជាក់លាក់ (តាមរយៈ Cloud Function/FCM)
- **ជជែក**: មើលនិងឆ្លើយតបសារជជែកជាមួយអ្នកប្រើប្រាស់ទាំងអស់ ជាមួយការជូនដំណឹងសារមិនទាន់អាន

## សេវាកម្មបច្ចេកទេសសំខាន់ៗ
- **AuthService**: គ្រប់គ្រងស្ថានភាពចូលប្រើ តួនាទី (admin/user) និងសិទ្ធិសមាជិកភាព
- **NotificationService**: ជូនដំណឹងតាម FCM និង Local Notification (រំលឹកសមាជិកភាពមុនផុតកំណត់ ៧ ថ្ងៃ ១ ថ្ងៃ និងថ្ងៃផុតកំណត់)
- **AnalyticsService**: កត់ត្រាព្រឹត្តិការណ៍ប្រើប្រាស់ (មើលមាតិកា មើលវីដេអូចប់ ចំណាំអត្ថបទ ការជាវ/ការបន្ត/ការផុតកំណត់ សមាជិកភាព ប្តូរភាសា/រចនាប័ទ្ម)
- **StorageService**: រក្សាទុកការកំណត់ (រចនាប័ទ្ម ភាសា ជីវមាត្រ ការជូនដំណឹង) និងបញ្ជីចំណាំក្នុងម៉ាស៊ីន
